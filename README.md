# Sales_Dashboard_PowerBI_Project

1. Project Title

Power BI Sales Dashboard Project

2. Short Description / Purpose

This repository contains a step-by-step Power BI project focused on building a Sales Dashboard from raw data to actionable insights. The purpose of this project is to teach how to import, transform, model, and visualize sales data in Power BI to help analyze performance metrics and support business decisions.

3. Tech Stack

The key technologies used in this project include:

-Microsoft Power BI Desktop
-Power Query (M language) for transforming and cleaning data
-DAX (Data Analysis Expressions) for calculated measures and KPIs
-Excel/CSV or database files as data source

4. Data Source

The data for this project comes from sales-related datasets, typically including fields such as:

-Product categories
-Sales transactions (dates, quantities, prices)
-Customer or regional sales data
-Revenue, profit, and discount information
-You will import this data into Power BI, clean and transform it using Power Query, and then use it for visual analysis.

5. Features / Highlights

-End-to-end sales dashboard creation using Power BI
-Importing and combining multiple data sources
-Data cleaning and transformation with Power Query
-Creation of calculated measures using DAX
-Interactive visuals like bar charts, line charts, slicers, and KPI cards
-Trend analysis for sales over time
-Category and region filtering for detailed analysis
-A professional, user-friendly dashboard ready for insights
